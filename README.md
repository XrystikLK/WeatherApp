# WeatherApp

Приложение для отображения погоды по заданным координатам (широте и долготе).

Для получения данных о погоде используется [OpenWeatherMap](https://openweathermap.org/)

## Функции
- Валидация введённых данных
- Отображение дополнительной информации о погоде
- Добавление погоды как виджет
- Интеграция с картой [OpenLayers](https://openlayers.org/) для отображения местоположения
  
Приложение размещено с помощью GitHub Pages.
